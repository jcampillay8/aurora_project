{% translate "Pronombres" %}

Descripción: Los pronombres personales se utilizan para referirse a personas o cosas sin nombrarlas directamente. Los pronombres posesivos indican pertenencia.
Objetivo: Enseñar el uso correcto de los pronombres personales y posesivos en oraciones.
Alcance: Uso de pronombres en contextos simples y cotidianos, tanto personales como posesivos.

{% translate "Tiempos Verbales" %}

Descripción: Los tiempos verbales son esenciales para expresar acciones en diferentes momentos. Incluyen: Presente Perfecto, Pasado Perfecto, Futuro Simple, Futuro Continuo, Present Continuous, Past Continuous, Future in the Past.
Objetivo: Enseñar a formar y usar correctamente estos tiempos verbales para describir acciones en el pasado, presente y futuro.
Alcance: Uso de diversos tiempos verbales en oraciones afirmativas, negativas e interrogativas.

{% translate "Conjunciones y Conectores" %}

Descripción: Las conjunciones subordinantes conectan una cláusula subordinada con una cláusula principal. Ejemplos incluyen: because, although, if.
Objetivo: Enseñar el uso de conjunciones subordinantes para conectar ideas de manera lógica.
Alcance: Uso de conjunciones subordinantes en oraciones compuestas para unir ideas relacionadas.

{% translate "Modal Verbs" %}

Descripción: Los verbos modales se utilizan para expresar posibilidad, obligación, permiso y otros matices. Ejemplos incluyen: may, might, must, shall, should, will, would.
Objetivo: Enseñar el uso de verbos modales en diferentes contextos.
Alcance: Uso de verbos modales en oraciones afirmativas, negativas e interrogativas para expresar diferentes significados.

{% translate "Phrasal Verbs" %}

Descripción: Los phrasal verbs son combinaciones de verbos con preposiciones o adverbios que adquieren un significado específico. Pueden ser separables e inseparables.
Objetivo: Enseñar el uso de phrasal verbs básicos en contextos cotidianos.
Alcance: Uso de phrasal verbs en oraciones para expresar diferentes acciones y significados.

{% translate "Reported Speech (Discurso Indirecto)" %}

Descripción: El discurso indirecto se utiliza para reportar lo que alguien más ha dicho, cambiando el tiempo verbal y los pronombres según sea necesario.
Objetivo: Enseñar a transformar oraciones directas en discurso indirecto, incluyendo los cambios de tiempo verbal y pronombres.
Alcance: Uso de discurso indirecto en oraciones para reportar declaraciones, preguntas y órdenes.

{% translate "Claúsulas de Relativo (Relative Clauses)" %}

Descripción: Las cláusulas de relativo proporcionan información adicional sobre un sustantivo. Pueden ser restrictivas (esenciales para el significado de la oración) o no restrictivas (adicionales y separadas por comas).
Objetivo: Enseñar a identificar y usar cláusulas de relativo restrictivas y no restrictivas.
Alcance: Uso de cláusulas de relativo para añadir información sobre personas y cosas en oraciones.

{% translate "Expresiones de Cantidad" %}

Descripción: Las expresiones de cantidad se utilizan para describir la cantidad de algo. Ejemplos incluyen: some, any, much, many.
Objetivo: Enseñar el uso correcto de expresiones de cantidad en diferentes contextos.
Alcance: Uso de expresiones de cantidad en oraciones afirmativas, negativas e interrogativas.

{% translate "Estructuras de Preguntas y Negaciones" %}

Descripción: Las preguntas indirectas y las "tag questions" son estructuras avanzadas para preguntar y confirmar información.
Objetivo: Enseñar a formular preguntas indirectas y "tag questions" para una comunicación más natural.
Alcance: Uso de preguntas indirectas y "tag questions" en conversaciones y escritos.
