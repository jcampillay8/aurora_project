{% translate "Pronombres Reflexivos" %}

Descripción: Los pronombres reflexivos se utilizan cuando el sujeto y el objeto de la oración son la misma persona o cosa. Ejemplos incluyen: myself, yourself, himself, herself, itself, ourselves, yourselves, themselves.
Objetivo: Enseñar el uso correcto de los pronombres reflexivos en oraciones.
Alcance: Uso de pronombres reflexivos en contextos variados para enfatizar o referirse al mismo sujeto.

{% translate "Tiempos Verbales: Pasado Simple" %}

Descripción: El pasado simple se utiliza para describir acciones que ocurrieron y finalizaron en un momento específico del pasado. Incluye verbos regulares e irregulares.
Objetivo: Enseñar a formar y usar el pasado simple con verbos regulares e irregulares.
Alcance: Uso del pasado simple en oraciones afirmativas, negativas e interrogativas.

{% translate "Preposiciones de Tiempo" %}

Descripción: Las preposiciones de tiempo indican cuándo ocurre algo. Ejemplos incluyen: at, on, in.
Objetivo: Enseñar el uso correcto de las preposiciones de tiempo en diferentes contextos.
Alcance: Uso de preposiciones de tiempo para describir momentos específicos o períodos de tiempo.

{% translate "Conjunciones y Conectores" %}

Descripción: Las conjunciones coordinantes conectan palabras, frases o cláusulas que tienen la misma función. Ejemplos incluyen: and, but, or.
Objetivo: Enseñar el uso de conjunciones coordinantes para conectar ideas de manera lógica.
Alcance: Uso de conjunciones coordinantes en oraciones compuestas para unir ideas relacionadas.

{% translate "Comparativos y Superlativos" %}

Descripción: Los comparativos se utilizan para comparar dos cosas, mientras que los superlativos se utilizan para destacar el grado más alto de una cualidad entre tres o más cosas. Incluye formas regulares e irregulares.
Objetivo: Enseñar a formar y usar comparativos y superlativos en inglés.
Alcance: Uso de comparativos y superlativos en oraciones para hacer comparaciones y destacar cualidades.

{% translate "Modal Verbs: Can, Could" %}

Descripción: Los verbos modales "can" y "could" se utilizan para expresar habilidad, posibilidad, permiso y solicitudes.
Objetivo: Enseñar a usar "can" y "could" en diferentes contextos para expresar habilidades, permisos y solicitudes.
Alcance: Uso de "can" y "could" en oraciones afirmativas, negativas e interrogativas.

{% translate "Gerundios e Infinitivos" %}

Descripción: Algunos verbos requieren que el verbo siguiente esté en forma de gerundio (verbo + ing) o infinitivo (to + verbo).
Objetivo: Enseñar a identificar y usar verbos que requieren gerundios e infinitivos.
Alcance: Uso de gerundios e infinitivos en oraciones para completar el significado de otros verbos.
